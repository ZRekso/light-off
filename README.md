**Description :**  
Light-OFF est un jeu de réflexion développé en Java. Le but est d'éteindre toutes les lumières de la grille. Ce projet a été réalisé dans le cadre du module IHM en BUT Informatique.

**Fonctionnalités :**
* Grille de jeu interactive.
* Interface graphique réalisée avec JavaFX et Scene Builder.
* Gestion du projet via l'outil de build Gradle.
